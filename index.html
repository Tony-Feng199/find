遇到的问题以及解决

遇到的问题

在起初时，本打算做一个动态的网站，但是由于遇见了较多问题，（如获取一个对象的key出错等很多涉及对象的基本问题）许多难以解决，又考虑到web项目的大小决定换成静态网站依据MDB模板进行开发，在使用MDB模板时也遇到了较多问题，大多是达不到我预期的效果，如在遮盖和弹性盒型布局下，将图片做成超链接的形式，超链接失效，图片排版设置不美观等，最后网页托管时也遇到一些小问题，如GitHub链接不上，在用git命令时，不能直接和GitHub项目相链接等问题，网页更新错误（访问出错等）

已经解决的问题

MDB达不到预期效果，通过上网找大佬的博客和自己的摸索直接使用内联样式解决了遮盖和弹性盒型布局下图片超链接失效的问题。GitHub托管网页，链接不上网络的原因，换网络就可以解决，本地仓库不能和GitHub项目相链接，通过设置git来设置key使GitHub识别本地仓库，网页更新出错， 用 git add * //把本地仓库的文件上传到缓存。git commit -m 'del' //把第一步上传到缓存的东西上传到本地仓库，其中'del'是操作标识，内容随便填，方便用户观看。 git push origin master //把本地仓库的文件上传到远程仓库。
